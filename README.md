# Frekama Fruits (フレカマ・フルーツ)

とっても可愛いフルーツたちをくっつけて大きくしていく、癒やし系のパズルゲームです！

## ゲームの特徴

- **可愛いキャラクター**: ぴんちゃんデザインの表情豊かなフルーツたちが全11種類登場します。
- **進化したUI**: フルーツの進化順番がひと目で分かる「エボリューションチャート」を搭載。
- **サバイバルモード**: ステージ制を廃止し、ひたすらハイスコアを目指すエンドレスモード。自己ベスト記録も保存されます。
- **癒やしの演出**: 応援キャラクターが動いたり、心地よい音楽（ON/OFF切り替え可能）とともにプレイできます。
- **レスポンシブ対応**: PCでもスマホでも、画面サイズに合わせて最適な表示で遊べます。

## 遊び方

1. フルーツをタップ（またはクリック）して落とします。
2. 同じ種類のフルーツがぶつかると、一回り大きなフルーツに進化します。
3. フルーツが箱から溢れないように注意しながら、究極の「スイカ」を目指してハイスコアを更新し続けましょう！

## 開発者向け情報

このプロジェクトは Vite を使用して構築されています。

### セットアップ

```bash
npm install
```

### 開発用サーバーの起動

```bash
npm run dev
```

### 本番用ビルド

```bash
npm run build
```
